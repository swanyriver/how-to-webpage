Make It Your Own

To use these scripts for yourself the first thing you will need to do is replace the the USERNAME and PASSWORD variables with your own credentials. If you like you can set them as command arguments or prompt for input for them. Or if you want to track your changes on a public repository then read them from a file on your local drive like so: password=$(cat ~/gitpass.txt). Another option is the store and retrieve these information in your git config files.

After that there are many improvements or personalizations you can make to these scripts:

    • set up the auto clone to be able to search other users repositories as well as your own
    • Add a flag that will cause the script to clone your most recently updated repository without prompting
    • Add the --silent flag to the curl calls to suppress the Progress Meter if you find it annoying.
    • Add more initialization activity on a successful remote repository creation.
    • Automate another part of your git/Github workflow!
